---

## 🚀 Deployment to Hostinger

### Files to Upload:
```
public_html/
├── index.html
├── contact.html
├── send-email.php          ⭐ Important!
├── assets/
│   ├── css/
│   ├── js/
│   └── images/
```

### Steps:
1. Log into Hostinger
2. Open File Manager
3. Go to `public_html/` folder
4. Upload all files
5. Done! ✅
